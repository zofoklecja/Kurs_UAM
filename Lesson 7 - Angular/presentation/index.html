More about scopes

Scopes, inheritance

Scopes are structured in to the inheritance tree

Scopes events


$scope.$on('eventName', function (event, arg1, arg2) {});

$scope.$emit('eventName', arg1, arg2);
//event will bubble up the scopes hierarchy

$scope.$broadcast('eventName', arg1, arg2);
//event will go down the scopes tree to the descendants

Factories, Services etc.

Various helpers, services, libs etc.


angular.module('myapp').factory('myHelpers', function () {
    return {
        helperFn: function () { ... }
    };
});

Dependency injection


angular.module('myapp').controller('MyCtrl', function ($scope, myHelpers) {
    myHelpers.helperFn();
});

Http resource


angular.module('myapp').factor('resource', function ($http) {
   return {
       loadTodos: function () {
            return $http.get('/api/todos');
       }
   }
});
