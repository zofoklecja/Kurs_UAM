BMI to współczynnik powstały przez podzielenie masy ciała podanej w kilogramach przez kwadrat wysokości podanej w metrach.

Oznaczyć zakresy wartości:

19 - 25 - oznaczyć wynik kolorem zielonym (klasa css ok)
17 - 30 - wartości niezbyt dobre (klasa css medium)
reszta - klasa css bad

Wartości należy odczytać ze sliderów w pliku HTML. Zadanie powinno być zrobione bez zmieniania samego pliku HTML!
